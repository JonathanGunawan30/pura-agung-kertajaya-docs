# Dashboard Unit Pura

Selamat datang di **Admin Panel Pura Agung Kertajaya**.

Halaman **Dashboard Utama** ini adalah pusat kontrol yang akan menyambut Anda setiap kali berhasil login. Di sini, Anda dapat mengakses berbagai modul untuk mengelola konten yang tampil di website utama Pura secara cepat dan mudah.

![Tampilan Dashboard Admin Pura](/img/pura/dashboard-pura.png)

## Penjelasan Antarmuka

Tampilan dashboard ini didesain minimalis untuk memudahkan Anda fokus pada pengelolaan konten. Berikut adalah penjelasan bagian-bagiannya:

### 1. Panel Akses Cepat (Quick Access)
Ini adalah area utama di tengah layar. Bagian ini berisi pintasan (*shortcut*) ke berbagai modul website. Anda cukup mengklik salah satu kotak untuk mulai mengedit konten.

Menu yang tersedia meliputi:
* **Aktivitas:** Mengelola berita atau agenda kegiatan pura.
* **Galeri Foto:** Menambah atau menghapus dokumentasi foto kegiatan.
* **Hero Slides:** Mengatur gambar *banner* utama yang muncul paling depan di website.
* **Kata Sambutan, Struktur Organisasi, Fasilitas, dll:** Mengubah informasi profil pura.

:::info Catatan
Setiap perubahan yang Anda lakukan pada menu-menu di atas akan **langsung berdampak** pada tampilan website publik Pura Agung Kertajaya.
:::

### 2. Sidebar Menu (Kiri)
Di sisi kiri terdapat navigasi utama untuk berpindah antar kategori besar:
* **Dashboard Utama:** Kembali ke halaman depan ini.
* **Pura:** Berisi submenu manajemen internal pura.
* **Global & Umum:** Pengaturan yang bersifat umum untuk website.
* **Ganti Password:** Fitur keamanan akun Anda.

### 3. Mode Tampilan
* **Mode Gelap/Terang:** Ikon bulan/matahari di pojok kanan atas untuk mengubah tema tampilan admin agar nyaman di mata.

---

:::tip Tips Navigasi
Jika Anda tersesat atau ingin kembali ke menu awal dengan cepat, cukup klik tombol **Dashboard Utama** berwarna oranye di menu sebelah kiri.
:::
