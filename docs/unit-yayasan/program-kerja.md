# Manajemen Program Kerja

Menu **Program Kerja** berfungsi untuk menayangkan rencana strategis, agenda rutin, atau target yang ingin dicapai oleh Yayasan.

Informasi ini akan tampil di halaman Tentang Yayasan sebagai poin-poin penting yang memudahkan umat atau masyarakat memahami fokus kegiatan Yayasan saat ini.

## Tampilan Publik
Di website, Program Kerja tampil berdampingan dengan foto ilustrasi. Poin-poin kegiatan akan disusun secara vertikal di sebelah kiri gambar.

![Tampilan Program Kerja di Website Publik](/img/yayasan/program-public.png)

## 1. Tampilan Admin
Berbeda dengan menu berita yang menampilkan banyak daftar, halaman ini hanya menampilkan **satu tampilan pratinjau (*preview*)** dari Program Kerja yang sedang aktif.

![Tampilan Program Kerja di Admin](/img/yayasan/program-list.png)

Di sini Anda bisa melihat gambar yang sedang digunakan beserta teks daftar program kerjanya.

## 2. Mengedit Konten
Untuk memperbarui daftar kegiatan atau mengganti gambar ilustrasi, klik tombol **Edit Program** berwarna oranye di pojok kanan atas.

Anda akan diarahkan ke formulir berikut:

![Formulir Edit Program Kerja](/img/yayasan/program-form.png)

### A. Foto Pendukung (Kiri)
* **Fungsi:** Sebagai ilustrasi visual yang muncul di sebelah kanan teks.
* **Upload:** Klik ikon pensil di sebelah kanan atas gambar untuk mengganti foto.

### B. Isi Program Kerja (Kanan)
Di kolom ini, Anda menuliskan daftar kegiatannya.

:::info Tips Penulisan Daftar
Kolom ini menggunakan format teks sederhana. Agar tampilan di website rapi menjadi daftar (list):
1.  Tulis satu kegiatan per baris.
2.  Gunakan tombol **Enter** untuk memisahkan antar poin kegiatan.
3.  Anda tidak perlu menulis nomor (1, 2, 3) secara manual, karena sistem website akan otomatis memberikan nomor atau simbol poin pada setiap baris baru.
:::

3. Setelah selesai merevisi, klik tombol **Simpan Perubahan**.
