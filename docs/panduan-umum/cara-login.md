# Cara Login ke Sistem

Untuk mengakses dashboard admin, ikuti langkah berikut:

1. Buka halaman login.
2. Masukkan **Email** yang sudah didaftarkan.
3. Masukkan **Password**.

![Tampilan Halaman Login](/img/login-screen.png)

4. Klik tombol **Masuk Dashboard**.

:::tip Tips Keamanan
Jangan pernah membagikan password Anda kepada orang lain!
:::
