# Profil Pengguna & Keamanan

Halaman ini berfungsi untuk mengelola keamanan akun Anda.

## Tampilan Halaman
Halaman ini terbagi menjadi dua bagian utama:
1.  **Info Pengguna (Kiri):** Menampilkan alamat email yang terdaftar sebagai identitas login Anda.
2.  **Ganti Password (Kanan):** Formulir untuk mengubah kata sandi lama menjadi yang baru.

![Halaman Ganti Password](/img/global/user-profile.png)

---

## Cara Mengganti Password

Ikuti langkah-langkah berikut jika Anda ingin mengubah kata sandi:

1.  **Akses Menu:** Klik menu **Ganti Password**
2.  **Isi Password Baru:**
    * Masukkan kata sandi baru yang Anda inginkan pada kolom **Password Baru**.
    * *Saran Keamanan:* Gunakan kombinasi huruf besar, huruf kecil, dan angka. Minimal 8 karakter agar tidak mudah ditebak.
3.  **Konfirmasi Password:**
    * Ketik ulang kata sandi baru yang sama persis pada kolom **Konfirmasi Password Baru**.
    * Ini untuk memastikan Anda tidak salah ketik.
4.  **Simpan:**
    * Klik tombol **Simpan Password** berwarna oranye di pojok kanan bawah.

:::info Info Login
Setelah password berhasil diganti, pada sesi login berikutnya Anda **WAJIB** menggunakan password baru tersebut. Password lama sudah tidak berlaku.
:::

:::tip Lupa Password?
Jika Anda lupa password dan tidak bisa login sama sekali, silakan hubungi **Administrator Pusat** atau tim pengembang website untuk melakukan *reset* password secara manual.
:::
