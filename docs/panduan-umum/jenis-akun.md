# Penjelasan Hak Akses

Sistem ini membagi pengguna menjadi **4 kategori** berbeda. Tampilan dan menu yang Anda lihat akan menyesuaikan otomatis dengan tanggung jawab Anda.

## 1. Administrator Pusat
Ini adalah akun level tertinggi yang memegang kendali penuh atas sistem.
* **Tanggung Jawab:** Manajemen seluruh unit dan konfigurasi sistem global.

## 2. Pengurus Unit Pura
Akun khusus untuk unit Pura.
* **Tanggung Jawab:** Manajemen unit pura.
* **Batasan:** Tidak bisa melihat data Yayasan dan data Pasraman.

## 3. Pengurus Unit Yayasan
Akun khusus untuk unit Yayasan.
* **Tanggung Jawab:** Manajemen unit yayasan.
* **Batasan:** Tidak bisa melihat data Pura dan data Pasraman.

## 4. Pengurus Unit Pasraman
Akun khusus untuk unit Pasraman.
* **Tanggung Jawab:** Manajemen unit pasraman.
* **Batasan:** Tidak bisa melihat data Pura dan data Yayasan.

---

:::info Mengapa Akses Dibatasi?
Pemisahan hak akses ini bertujuan untuk **menjaga keamanan data**. Sistem memastikan setiap pengurus hanya fokus pada wilayah kerjanya masing-masing dan mencegah perubahan data yang tidak disengaja pada unit lain.
:::
